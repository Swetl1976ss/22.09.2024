Git

  • git -v Эта команда выведет версию Git
  • git clone ссылка с GitHub клонирует содержимое репозитория
  • git status отслеживание изменений в репозитории
  • git add название файла - добавляет файл в отслеживаемые
  • git commit -m "название коммита" создает коммит, показывает изменение в файле
